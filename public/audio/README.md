# public/audio/

Drop the two adaptive-music loops here:

- `music-calm.ogg` — calm layer (few bugs)
- `music-intense.ogg` — intense layer (many bugs)

They crossfade by the bug meter at runtime. Same BPM + length, seamless loops.
See [../../docs/MUSIC.md](../../docs/MUSIC.md) for full spec, prompts and
licensing notes. Until the files exist, the game simply plays no music.
